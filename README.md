# AquaTrack Logistics – IoT-Powered Fish Transportation Monitoring
AquaTrack Logistics APP is a hackathon project developed in 2024 to address challenges in Uganda's fish transportation supply chain through digitalization and IoT technology. As part of a multidisciplinary team, I contributed as the application developer, building a mobile application using Andriod Studio and Flutter SDK to digitized paper-based fish inspection processes and provided a visual interface for tracking IoT devices in real time.

## Key Features
* Digital fish inspection forms to replace manual paperwork
* Real-time visualization of IoT device location using GPS tracking
* Temperature monitoring for cold-chain compliance
* Improved traceability, transparency, and operational efficiency

## Project Impact
The solution aimed to reduce post-harvest losses, improve regulatory compliance, and enhance the quality assurance process for fish transportation in Uganda by combining digital inspections with IoT-powered cold-chain monitoring.
